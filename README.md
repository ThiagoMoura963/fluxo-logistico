# Fluxo Logístico

Sistema WEB Mobile-First para gestão operacional logística e programação de cargas.

## Sobre o projeto

O Fluxo Logístico é um sistema desenvolvido para substituir controles operacionais realizados manualmente via WhatsApp, centralizando programações, cargas, horários, reservas e atualizações operacionais em uma interface simples, rápida e otimizada para uso diário.

O foco principal do projeto é reduzir retrabalho operacional, automatizar processos repetitivos e melhorar a organização da operação logística em tempo real.

---

## Objetivos

- Reduzir digitação manual
- Automatizar geração de programações
- Facilitar atualização de cargas e saldos
- Melhorar visualização operacional no celular
- Centralizar informações da operação
- Integrar com WhatsApp
- Melhorar produtividade operacional

---

## Stack inicial

- Next.js
- PostgreSQL
- TailwindCSS
- Evolution API
